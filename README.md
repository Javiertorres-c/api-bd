API BD Javier - Gestión de Alumnos

API REST desarrollada con Spring Boot usando PostgreSQL como base de datos y UUID como identificador principal.

Tecnologías utilizadas

- Java 17+
- Spring Boot
- Spring Web
- Spring Data JPA
- PostgreSQL
- Hibernate


Requisitos previos

Antes de ejecutar el proyecto asegúrate de tener instalado:

- Java 17 o superior
- PostgreSQL
- Postman- opcional

Base de datos
